6.3 .loc & .iloc

說明 pandas
取[某列,某行]，得value (此時不再是df type) df.loc["列名1","行名2"]
若要維持df type，要加中括號 df.loc[["列名1"],["行名2"]]
取多個列，多個行 df.loc[["列名1","列名2"],["行名2","行名2"]]
若要取多個列就好，行我全要 
錯誤: df.loc[[列名1, 列名2], :]

正確: df.loc[[列名1, 列名2]]
若要取多個行就好，列我全要 df.loc[:, [行名1, 行名2]]
取連續的列 
錯誤: df.loc[["列名2":"列名7"]]

正確: df.loc["列名2":"列名7"]
排序後，取連續的行 
錯誤: df.loc[:, ["行名2":"行名7"]]

正確: df.loc[:, "行名2":"行名7"]
.iloc就是把列名全改成列數(0開始), 行名全改成行數(0開始)就好 
df.iloc[0,2] # 返回value
df.iloc[[0],[2]] # 返回df type
df.iloc[[0,2], [0,1]] #多列多行
df.iloc[[0,2]] # 多列
df.iloc[:, [0,1]] # 多行

6.4 indexing (another filter & arrange method)

  • 簡單講，就是把某個column，設成row index，那接下來就可以用.loc[[列名1, 列名2]] 來進行filter

  • 更特別的是，row index可以接受multi-column，所以當我們想篩選出類似實驗設計的多種treatment時(例如”A=1且B=2” 或 “A=2且B=3”)，特別適合用這種方式來篩選。

說明

將欄位 a 設成 index

將 index 回覆成欄位

df_ind = df.set_index(“a”)

df_orig = df_ind.reset_index()

 
df %>%
  filter(a == 1)
 
df \
  .set_index("a") \
  .loc["1"] \
  .reset_index()
 
df %>%
  filter(a %in% c("1","2"))
 
df \
  .set_index("a") \
  .loc[["1","2"]] \
  .reset_index()
 
df %>%
  filter(
    (a == 1 & b == 2) |
    (a == 3 & b == 5)
  )
 
df \
  .set_index("a","b") \
  .loc[(1,2), (3,5)] \
  .reset_index()
 
df %>%
  arrange(a)
 
df \
  .set_index("a") \
  .sort_index() \
  .reset_index()
 
df %>%
  arrange(desc(a))
 
df \
  .set_index("a") \
  .sort_index() \
  .reset_index(ascending = False)
 
df %>%
  arrange(a, desc(b))
 
df \
  .set_index("a", "b") \
  .sort_index(
    level = ["a","b"],
    ascending = [True, False]
  )

6.6.2 對 dataframe 做 summarise (結果仍是df)

dplyr Pandas 
df %>%
  summarise(mean_col1 = mean(col1)) #仍是 table
 
# series
df[['col1']].mean()

# dataframe
df[['col1']].mean()

df[['col1']].mean() \
  .to_frame(name = "mean_col1")

df[['col1']].agg(np.mean) \
  .to_frame(name = "mean_col1")

df \
  .agg({'col1': np.mean}) \
  .to_frame(name = "mean_col1")
 
df %>%
  summarise(
    mean_col1 = mean(col1),
    med_col1 = median(col1)
  )
 
df \
  .agg(
    {
      'col1': np.mean,
      'col2': np.median
    }
  ) #沒辦法命名了，格式是直的
 
df %>%
  summarise(
    mean(col1),
    mean(col2),
    median(col2)
  )
 
df \
  .agg({
    'col1': np.mean,
    'col2': [np.mean, np.median]
  }) # 格式變成 cross
 
range2 = function(x){
  max(x) - min(x)
}

summarise(df, mean(col1), median(col2), range2(col1))
 
def range2(x):
  return x.max() - x.min()

df.agg([np.mean, np.median, range2])

6.8 pivot table

  • pandas有個dplyr沒有的優點，就是他可以做excel的樞紐分析表
  • 最簡單的例子，就是你可以指定table的row是性別，col是年級，然後cell是去算該性別該年級下的平均成績之類的
dplyr無, Excel樞紐 pandas 
df.pivot_table(
  index = cat_col1, #row
  columns = cat_col2, #col
  values = con_col, #cell要by誰做計算
  aggfunc = np.mean, #計算用的function
  margins = True #是否要算overall row/col
)
